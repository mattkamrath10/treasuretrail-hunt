PHASE 1 — MONETIZATION FOUNDATION IMPLEMENTATION

We are officially implementing the first monetization layer for TreasureTrail.

GOAL:
Create a SIMPLE, STABLE, production-ready monetization foundation before submitting the app to Flippa.

We are using a 3-tier structure:

1. FREE USER
2. BOOSTED EVENT / BOOSTED POST ($3 one-time)
3. TREASURETRAIL PRO ($9/month)

IMPORTANT:
This phase is architecture + gating + UI preparation.
DO NOT build fake placeholder premium systems.
DO NOT over-engineer.
Keep everything mobile-first and stable.
