Improve the Live Events page layout and scrolling behavior so the primary action buttons remain visible and accessible while the events feed scrolls independently.

PROBLEM

BUT the empty-state section and "Upload an Event" button are inside the scrolling content area

Once many events exist, the Upload Event button gets pushed far down the page and becomes difficult to access.

This hurts:

user engagement
