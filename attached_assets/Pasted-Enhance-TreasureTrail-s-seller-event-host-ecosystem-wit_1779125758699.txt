Enhance TreasureTrail’s seller/event-host ecosystem with safer location sharing, scout coordination, and marketplace logistics features.

========================================
GOAL
====

Make TreasureTrail highly attractive to:

* yard sale hosts
* estate sale companies
* auction houses
* local sellers
* storage locker sellers
* marketplace flippers
* Whatnot/live sellers

The platform should help sellers:

* reach more buyers
* coordinate pickups
* use scouts
* safely share locations
* increase exposure beyond local audiences

WITHOUT exposing unsafe private information publicly.

========================================

1. REQUIRED GENERAL LOCATION
   ========================================

For ALL:

* listings
* live events
* auctions
* estate sales
* yard sales
* marketplace submissions

Require ONE of the following:

* ZIP code
  OR
* City + State

Examples:

* Phoenix, AZ
* 90210
* Brooklyn, NY

This becomes:

* searchable
* filterable
* scout-compatible
* visible publicly

DO NOT require exact addresses publicly.

========================================
2. OPTIONAL EXACT LOCATION
==========================

Add an OPTIONAL field:

"Exact Pickup Address"

This should remain:

* hidden by default
* private
* only visible after seller approval/contact

Add optional toggle:
"Reveal exact location after contact"

OR

"Address shared after appointment/purchase"

========================================
3. LOCATION/PICKUP TYPES
========================

Add selectable logistics/pickup badges:

* Local Pickup
* Shipping Available
* Meetup Only
* Scout Delivery Available
* Nationwide Shipping
* Appointment Required

Display these badges on:

* event cards
* listing cards
* item detail pages

========================================
4. SCOUT INTEGRATION
====================

Add seller-facing scout tools.

New optional toggles:

* Need a Scout
* Scouts Available Nearby
* Pickup Assistance Available

If enabled:

* nearby scouts can respond
* buyers can coordinate pickups
* scouts can offer transport/shipping help
